## Your 10-Minute Sprint Formula

1. **Read** the "Why This Matters" section (2 min)
2. **Copy** one code block that looks useful (1 min)
3. **Run** it in Python (2 min)
4. **Change** one value and run again (2 min)
5. **Write** one line in your own words about what it does (3 min)

---

## After Phase 1 — You Can:

- Write scripts that store and manipulate data
- Process lists of servers, configs, or log entries
- Make decisions with if/elif/else
- Automate repetitive tasks with loops
- Build reusable functions other scripts can call
- Read JSON API responses (they're just dicts)
- Format professional output with f-strings
